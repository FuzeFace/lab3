
# Лабораторная работа 3

## Цель работы

Получить практический опыт работы с мультимодальными моделями на задаче саммаризации короткого видео.

## Исходные данные

В качестве исходных данных был выбран YouTube-ролик длительностью до 10 минут. Видео посвящено объяснению различий между искусственным интеллектом, машинным обучением, глубоким обучением и генеративным ИИ.

## Использованный пайплайн

В работе был реализован следующий пайплайн:

YouTube-видео → Whisper → транскрипт → PySceneDetect → ключевые кадры → BLIP → описания кадров → итоговое структурированное саммари.

## Ход работы

Сначала видео было скачано и сохранено в рабочую папку Google Colab. Затем аудиодорожка была обработана моделью Whisper, которая вернула транскрипт с таймкодами.

После этого был использован PySceneDetect для поиска смен сцен. Так как явных смен сцен обнаружено не было, видео было равномерно разделено на 8 временных фрагментов. Из середины каждого фрагмента был извлечён ключевой кадр с помощью ffmpeg.

Затем каждый ключевой кадр был передан в VLM-модель BLIP. Модель сгенерировала текстовые описания изображений. После этого для каждой сцены был сформирован объединённый контекст: таймкод, описание ключевого кадра и соответствующий фрагмент транскрипта.

На последнем этапе на основе полученного мультимодального контекста было сформировано структурированное саммари с таймкодами.

## Полученные результаты

В результате выполнения работы были получены следующие файлы:

transcript.txt — транскрипт видео с таймкодами.

scenes.json — список временных фрагментов и ключевых кадров.

frame_descriptions.json — описания ключевых кадров, полученные с помощью BLIP.

scene_context.json — объединённый мультимодальный контекст.

summary.md — итоговое структурированное саммари.

## Вывод

В ходе лабораторной работы был реализован мультимодальный пайплайн для анализа короткого видео. Whisper позволил получить текстовую информацию из аудио, BLIP добавил визуальные описания кадров, а итоговое саммари объединило информацию из разных модальностей. Такой подход даёт более полное представление о содержании видео, чем обычная текстовая саммаризация.
